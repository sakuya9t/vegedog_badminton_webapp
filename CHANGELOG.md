# Changelog

## [1.0.5] - 2026-06-08
- 付款提醒改为每日发送，直到完成付款为止
- 赛后总结导航图标更新
- 新增定时 ping 防止开发数据库休眠

## [1.0.4] - 2026-05-22
- 新增「赛后总结」页面：可添加餐厅推荐，支持通过 Google Maps 链接自动填充店名、地址、营业时间等信息
- 餐厅支持标签筛选、推荐/不推荐投票及随机抽取功能
- 新增贡献者：茶茶不吃饭、Vega

## [1.0.3] - 2026-05-11
- 新建接龙时通知关注者改为可选，样式优化为卡片形式
- 付款状态整合进报名队列，管理员可直接查看每位成员的付款情况
- 锁定后队列标题新增已付款人数统计
- 搜索参与者时可直接在结果中切换加时状态
- 修复付款提醒定时任务未能正常发送邮件的问题

## [1.0.2] - 2026-05-02
- 修复参与记录热力图月份标签重叠问题，优先显示较晚月份
- 微信内置浏览器打开时提示用户切换到系统浏览器
- 接龙结束24小时后自动向未付款成员发送付款提醒邮件

## [1.0.1] - 2026-05-02
- 改动记录新增「加入接龙」时间日志
- 修复发给球馆邮件星期显示错误（时区问题）
- 修复截止后退出警告在有候补时误触发
- +时操作新增确认弹窗
- 历史页改为仅显示管理员手动移入历史的接龙

## [1.0.0] - 2026-05-01
- 截止时间后退出且人数将不足时，弹窗警告用户仍需分摊场地费
- 有候补时退出不触发警告，候补递补后自动发送邮件通知
- 发给球馆邮件标题动态填入星期，正文动态填入场地数

## [0.9.5] - 2026-04-20
- 管理员可编辑未锁定接龙的标题、地点、时间、场地数、人数上限、注意事项

- 所有邮件发送由 `ENABLE_EMAIL=true` 环境变量统一控制，防止测试环境误发

- 数据库：`withdraw_participant` 禁止从已锁定或已关闭接龙退出
